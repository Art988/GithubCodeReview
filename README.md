# Projeto - GithubCodeReview
**Status do Projeto:** Concluído. <br>
**Contribuidores:** Arthur Roberto, Breno Santos Leal, Daniel Zanqueta, Davi Alcanfor, Gabriel Peres. <br>
**Objetivo:** O projeto tem o objetivo de apresentar os países que os membros do grupo desejam visitar, além do motivo e curiosidades. Atualmente contém: Australia, Chile, Amsterdam, Paris. <br>
<br>
**Como Usar:** Entrar no link do GitHub Pages e poderá visualizar o site. <br>
**Como Contribuir:** Configure o respositório como origin na sua IDE e adicione o seu país desejado, seguindo o modelo das entradas anteriores. <br>
<br>
**Funcionalidades:** Página responsiva com scroll animado para as caixas de texto. <br>
**Tecnologias Utilizadas:** HTML, CSS e JavaScript. <br>
